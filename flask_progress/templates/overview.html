Project Overview

Introduction

According to the CDC, cardiovascular disease refers to several types of conditions affecting the heart and blood vessels. Common cardiovascular diseases and conditions include heart disease, stroke and hypertension (high blood pressure). Heart disease and stroke are two of the leading causes of death in the United States, major causes of disability and the principal causes of cardiovascular disease death [1], [2].

Identifying factors related to common cardiovascular diseases would enable targeted preventative health intervention for populations at highest risk to potentially reduce poor health outcomes and associated medical costs.

This project aims to investigate factors that lead to cardiovascular disease as well as to explore which supervised machine learning methods most accurately classify patients as having or not having cardiovascular disease given certain patient measurements and demogrpahic information.

Keywords: Machine Learning, Cardiovascular Disease, Diagnosis, Classification

Project Process

Step 1: Research / Planning / Project Setup
  • Research project topics and locate datasets appropriate for machine learning
  • Define project workflow and conceptualize data pipeline [3].
  • Create a repository in github, add README.md file to repository, upload initial dataset
Step 2: Create a Jupyter Notebook for ETL and Machine Learning
  • Data Cleaning: load dataset, inspect basics; handle missing data; handle outliers; binary encoding; explore our target variable’s balance
  • Data Exploration to decide which variables to keep: View distributions of variables, View descriptive stats, Check relationships among variables; Split data into training and testing, train decision tree and random forest to evaluate feature importance
  • Train different supervised classification machine learning models to predict diagnosis: logistic regression, random forest, Support Vector Machines (SVM)
  • Export chosen model for flask app
Step 3: create flask app for classification dashboard and deploy with Heroku

Dataset Features

The stroke dataset used for our Cardiovascular Disease Classifier can be found here.

Feature List:
  • Average Glucose Level: Number between 0 and 300 (mg/dL)
  • BMI: Number between 0 and 100
  • Age: Integer between 0 and 100
  • Ever married? Yes or No
  • Gender: Male or Female
